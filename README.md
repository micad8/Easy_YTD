#### **Download Youtube Video 😀👍**

###### A simple Windows tool designed for beginners like myself for downloading videos and audio from YouTube and other sites.
###### This is pretty much just yt-dlp and ffmpeg along with the code to allow beginners to use yt-dlp



**How To Install**

1. Go to the Releases page
2. Download the latest **Setup.exe** and run it, OR
3. Click the green **Code** button → **Download ZIP** and extract it



**How To Use**

* Double-click Extract Video to download a video as .mp4
* Double-click Extract Audio to download audio only as .mp3
* Paste the URL when prompted and press Enter
* Your file will appear in the Output folder when done
